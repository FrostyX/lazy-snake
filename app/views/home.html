Lazy Snake

A web based, easy to use and nice looking python profiler. You can post your cProfile results via url, file or terminal client. See live demo to see what Lazy Snake provides to you.

Terminal client

For frequent use, uploading profiler results manully, can be quite inconvenient. Its where terminal client comes handy. Easy to install, easy to use.

# Install
sudo pip install lazy-snake

# Usage
lazy-snake path/to/some/script.py

The lazy-snake command will profile your script via cProfile send them to the server and provides you link to your result.

Own server

As mentioned above, please feel free to run your own instance of Lazy Snake on your computer.

# Install
sudo pip install lazy-snake-server

# Run
lazy-snake-server

# Tell client to use your server (optional)
echo "URL = http://127.0.0.1:5000/" >> ~/.config/lazy-snake/client
